# 🚀 NASA Project

A complete space weather application built with **Flutter** (Frontend) and **Django Rest Framework** (Backend).

---

## 🧩 Project Overview
This project provides interactive visualizations and real-time space data powered by NASA APIs.  
The **Flutter app** serves as the user interface, while the **Django REST API** handles data processing and backend logic.
